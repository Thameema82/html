Inserting Images



An image can be inserted in an image using<img >using tag.The image to be displayed can be specified using the src attribute. Image can be anywhere in the current directory,else where in the web server,or in a different machine itself.

Changing the size of Image


Size of an image can be altered using height and weight attributes .We can specify the size in pixels.Note that the first image is having a border also .This is specified using the border attribute - border width is also specified in pixels.
